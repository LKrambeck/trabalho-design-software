# Meu Cantinho

Para a execução do projeto é necessário que se tenha **Docker** instalado. Com este software presente, faça:
1. Clone o repositório;
2. Rode na pasta raíz do projeto
    ```bash
    docker-compose build
    ```

3. Em seguida, execute
    ```bash
    docker compose up
    ```

Isso fará com que as rotas fiquem disponíveis em
```
http://127.0.0.1:8000/
```

Para acessar a documentação das APIs, entre em
```
http://127.0.0.1:8000/docs#/
```
